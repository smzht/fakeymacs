#+STARTUP: showall indent

* Fakeymacs extension - micro_key -

** ■ Micro Editor 用のキーの設定を行う

Micro Editor 用のキーの設定を行う拡張機能です。

*** コンフィグレーションパラメータ

|-------------------------+-------------------------------------------------------------------------|
| Configuration parameter | Description                                                             |
|-------------------------+-------------------------------------------------------------------------|
| fc.micro_target         | Micro Editor 用のキーバインドを利用するアプリケーションソフトを指定する |
|-------------------------+-------------------------------------------------------------------------|

*** 留意事項

● 本フォルダにある _config_personal.py を config_personal.py というファイル名にすることで
機能拡張ファイルとして機能します。micro_key Extension の機能を拡張する際にご利用ください。

● トップページの README にある次の項目も確認してください。

- [[/README.org#windows-terminal-の対応について]]
- [[/README.org#micro-editor-の対応について]]

上記の２つ目に記載していますが、Fakeymacs は Micro Editor が set clipboard terminal
の設定で動作するように調整しています。ただし、この設定では、マルチカーソルによるコピペが
正常に動作しないため、本件は制約事項とさせていただきます。
set clipboard internal の設定で正常に動作することは確認していますが、その場合は Windows
クリップボードとの相互利用ができなくなります。
また、Windows Terminal や Fakeymacs の設定の追加調整が必要となります。

● Windows Terminal で Micro Editor を利用する際は、設定 -> 二番目にある操作 メニューより、
「ウィンドウのサイズを変更する 上」の Alt+Shift+Up と 「ウィンドウのサイズを変更する 下」の
Alt+Shift+Down を削除してご利用ください。
