* Fakeymacs extension - mc_key -

** ■ Midnight Commander 用のキーの設定を行う

Midnight Commander 用のキーの設定を行う拡張機能です。

*** コンフィグレーションパラメータ

|-------------------------+-------------------------------------------------------------------------------|
| Configuration parameter | Description                                                                   |
|-------------------------+-------------------------------------------------------------------------------|
| fc.mc_target            | Midnight Commander 用のキーバインドを利用するアプリケーションソフトを指定する |
|-------------------------+-------------------------------------------------------------------------------|

*** キーバインド

**** ● 「カーソル移動」のキー設定

|---------+----------------------+-------------|
| Keybind | Function             | Description |
|---------+----------------------+-------------|
| M-<     | beginning_of__buffer |             |
| M->     | end_of_buffer        |             |
|---------+----------------------+-------------|

**** ● 「パネル操作」のキー設定

|---------+--------------------+-------------|
| Keybind | Function           | Description |
|---------+--------------------+-------------|
| Ctl-x o | other_window       |             |
|---------+--------------------+-------------|

*** 留意事項

● 本フォルダにある _config_personal.py を config_personal.py というファイル名にすることで
機能拡張ファイルとして機能します。mc_key Extension の機能を拡張する際にご利用ください。

● トップページの README にある次の項目も確認してください。

- [[/README.org#windows-terminal-の対応について]]
- [[/README.org#midnight-commander-の対応について]]

● mc から エディタを起動する際、Microsoft Edit のようにウィンドウタイトルを変更する機能を
持たないエディタは、settitle コマンドを経由して起動するようにしてください。
mc_key 拡張機能で設定しているキーバインドを無効とするための対策となります。
vim や emacsclient などが該当します。次のような alias を設定しておくのがお勧めです。

#+BEGIN_EXAMPLE
alias vi='settitle vim'
alias ec='settitle emacsclient -t'
#+END_EXAMPLE
