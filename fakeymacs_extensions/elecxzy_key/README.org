* Fakeymacs Extension - elecxzy_key -

** ■ elecxzy 用のキーの設定を行う

elecxzy 用のキーの設定を行う拡張機能です。

fc.use_esc_as_meta 変数または fc.use_ctrl_openbracket_as_meta 変数が True の場合に、Esc または C-[
キーが Meta キーとして機能するようになります。Esc を発行したい場合は、キーを二回押下してください。

*** コンフィグレーションパラメータ

|-------------------------+--------------------------------------------------------------------|
| Configuration parameter | Description                                                        |
|-------------------------+--------------------------------------------------------------------|
| fc.elecxzy_target       | elecxzy 用のキーバインドを利用するアプリケーションソフトを指定する |
|-------------------------+--------------------------------------------------------------------|

*** 留意事項

● 本フォルダにある _config_personal.py を config_personal.py というファイル名にすることで
機能拡張ファイルとして機能します。mc_key Extension の機能を拡張する際にご利用ください。
