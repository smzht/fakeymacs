#+STARTUP: showall indent

* Fakeymacs extension - vim_key -

** ■ Vim 用のキーの設定を行う

Vim 用のキーの設定を行う拡張機能です。Vim のキーバインドを活かしながら、Emacs のキーバインドが
追加で利用可能となります。

*** 環境設定

本拡張機能を使うために、次の設定を行ってください。

**** 1) 本拡張機能を有効にする

Keyhac のトップフォルダにある config_personal.py ファイルにおいて、vim_key 拡張機能の設定箇所
を有効（if 0: を if 1: に変更）にしてください。

**** 2) 必要に応じて config_personal.py ファイルを作成する

必要に応じて、本フォルダにある _config_personal.py を config_personal.py というファイル名
に複写して作成してください。
config_personal.py を作成することで、本拡張機能の個人設定ファイルとして機能します。

**** 3) Vim の設定を行う

Vim の場合、~/.vimrc の最後に次の設定を追加してください。

#+BEGIN_EXAMPLE
set nocompatible
set title
set hidden
set number
set ambiwidth=double
set selection=exclusive
set virtualedit=onemore
set whichwrap=b,s,h,l,<,>,[,]
set incsearch

inoremap <C-j> <C-o>
#+END_EXAMPLE

Neovim の場合、~/.config/nvim/init.lua の最後に次の設定を追加してください。

#+BEGIN_EXAMPLE
vim.opt.title = true
vim.opt.hidden = true
vim.opt.number = true
vim.opt.ambiwidth = "double"
vim.opt.selection = "exclusive"
vim.opt.virtualedit = "onemore"
vim.opt.whichwrap = "b", "s", "h", "l", "<", ">", "[", "]"
vim.opt.incsearch = true

vim.keymap.set("i", "<C-j>", "<C-o>")

vim.api.nvim_create_autocmd("VimLeave", {
  callback = function()
    os.execute("echo -ne '\\033]0;\\007'")
  end,
})
#+END_EXAMPLE

※ Windows 版 Vim の設定ファイルは、%LOCALAPPDATA% 以下に格納してください。

※ 設定にあるキーマップの設定は、インサートノーマルモードに移行するキーを追加するための設定です。
Vim を Midnight Commander と連携して使う場合、Vim のインサートノーマルモードに移行するための標準キー
C-o は利用できません。
このため、以降で説明するコンフィグレーションパラメータ fc.vim_insert_normal_mode_key に "C-j"
を設定することで、この問題を回避できるようにしています。
Midnight Commander を利用しないのであれば、この設定は不要となります。

※ Neovim の設定にある nvim_create_autocmd の設定は、Neovim 終了時にターミナルのタイトルを
初期化するための設定です。Vim には、終了時にタイトルを「Thanks for flying Vim」に置き換える機能が
ありますが、Neovim にはそのような機能がないため、この設定を追加しています。

*** コンフィグレーションパラメータ

|-------------------------------+---------------------------------------------------------------------------------|
| Configuration parameter       | Description                                                                     |
|-------------------------------+---------------------------------------------------------------------------------|
| fc.vim_target                 | Vim 用のキーバインドを利用するアプリケーションソフトを指定する                  |
| fc.vim_keep_in_insert_mode    | インサートモードで C-g を押下した際、インサートモート居続けるかどうかを指定する |
| fc.vim_insert_normal_mode_key | インサートノーマルモードに移行するためのキーを指定する                          |
|-------------------------------+---------------------------------------------------------------------------------|

※ fc.vim_keep_in_insert_mode 変数を True にすると、インサートモードで C-g を押下しても
ノーマルモートに移行しなくなります。ノーマルモートに移行するためには、Esc または C-[
を押下してください。

※ Vim を Midnight Commander と連携して使う場合は、fc.vim_insert_normal_mode_key 変数に
"C-j" を設定してご利用ください。

*** 留意事項

● 本拡張機能は、Vim のエキスパートの利用は想定していません。Emacsキーバインドユーザが環境条件
により Vim を使う必要が出た場合に、Vim の基本機能を Emacs キーバインドで操作できるようにするために
作成したものです。Vim の拡張機能に対しての動作は確認しておりませんので、ご了承ください。

● 本拡張機能では、IMEの状態を自動で変更する場合があります。このため、テキスト カーソル インジケータ
を有効とし、IME の状態を色で表示することをおすすめします。テキスト カーソル インジケータを有効と
する方法は、次を参照してください。

-  https://faq.nec-lavie.jp/qasearch/1007/app/servlet/relatedqa?QID=022081

また、Fakeymacs のトップフォルダにある config_personal.py 内の fc.use_ime_status_cursor_color
変数を True にする必要があります。

● Windows Terminal でテキスト カーソル インジケータを使っていると、表示位置が 2行くらいずれる場合が
あります。この場合は、新しいターミナルをタブで一度開くと是正されるようです。

● ノーマルモードで C-Enter と打つと、インサートモードに移行した後、Enter が入力される仕様としています。

● ノーマルモード、ビジュアルモード、インサートノーマルモードで日本語を入力すると、インサートモードに
移行した後、日本語が入力される仕様としています。

● ssh の接続先で Vim を使う場合、接続先に上記で紹介した Vim の設定を行えさえすれば、接続元で
本拡張機能が有効となり、Emacs キーバインドが利用できるようになります。（設定の中の title の有効化
により、ターミナルのタイトルが変更されることで実現しています。）

● Emacs キーバインドで実行されるコマンドは、Vim のノーマルモードでもインサートモードでも動作するよう
に作成しています。ただし、文字の入力をする場合は、Vim のキー（i や a など）でインサートモードに
明示的に移行する必要があります。

● 本拡張機能では、Vim のいくつかのモードの状態を変数で管理しています。このモードの状態と変数の設定が
ずれると正常に動作しませんので、そのような状態が発生した場合は Esc を押下して状態の初期化を行って
ください。

● モードの状態と変数の設定がずれる可能性高いコマンドとして c コマンドがあります。c コマンドは、
以降に motion や textobject を指定するための１つ以上のキーが入力されて初めてインサートモードに移行
しますが、本拡張機能では、機能の簡略化のために、c キーを入力した直後にインサートモードに入るもの
として変数の設定を行っています。
しかし、c コマンドの後にmotion や textobject を指定するためのキーが入力されなかった場合には
インサートモードに移行しないため、モードの状態と変数の設定にずれが発生する場合があります。
このような場合は Esc を押下して状態の初期化を行ってください。

● トップフォルダの config_personal.py ファイル内で fc.vim_keep_in_insert_mode を True に設定すると、
インサートモードで C-g を押下してもノーマルモードに移行しなくなります。
この場合にノーマルモードに移行するには、Esc や C-[ を押下するようにしてください。
なお、タブ操作のコマンドなど、コマンド実行時に強制的にノーマルモードに移行するようにしているものも
あります。その場合にインサートモードに移行するには、改めてインサートモードに入るコマンド（i や a など）
を入力する必要があります。

● インサートモードでノーマルモードのコマンドを実行したい場合には、C-A-o の押下で一回だけ実行可能
となります。

● 本拡張機能を Midnight Commander と共に利用する場合、C-o のキーがバッティングして正常に動作しません。
この場合は、トップフォルダの config_personal.py ファイル内で fc.vim_insert_normal_mode_key 変数に
"C-j" を設定してご利用ください。
