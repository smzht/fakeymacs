#+STARTUP: showall indent

* Fakeymacs extension - fresh_key -

** ■ Fresh Editor 用のキーの設定を行う

Fresh Editor 用のキーの設定を行う拡張機能です。

*** コンフィグレーションパラメータ

|-------------------------+------------------------------------------------------------------------------------------------------------------------------------------------------|
| Configuration parameter | Description                                                                                                                                          |
|-------------------------+------------------------------------------------------------------------------------------------------------------------------------------------------|
| fc.fresh_language       | Fresh Editor に設定している表示言語を指定する（US: English、JP: 日本語）                                                                             |
| fc.fresh_target         | Fresh Editor 用のキーバインドを利用するアプリケーションソフトを指定する                                                                              |
| fc.fresh_command_dict   | Fresh Editor の Command pallet で利用するコマンドを指定する（key: フルスペル英語コマンド、value[0]: 短縮英語コマンド、value[1]: 短縮日本語コマンド） |
|-------------------------+------------------------------------------------------------------------------------------------------------------------------------------------------|
